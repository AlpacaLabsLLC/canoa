# Room types and their typical FF&E categories

A reference for `/canoa-source-room` to pick which catalog categories to search per room type. Not exhaustive — read the brief first, use this only as a default.

## Hospitality

### Lobby

- **Seating** (lounge chairs, sofas, ottomans)
- **Tables** (coffee, side, console)
- **Lighting** (pendants, floor lamps, table lamps)
- **Casegoods** (reception desk, credenzas, sideboards)
- **Accents** (rugs, mirrors, artwork holders, planters)

### Guest room

- **Beds** (frame, headboard, nightstands)
- **Seating** (desk chair, lounge chair, bench)
- **Tables** (desk, side tables)
- **Lighting** (bedside, desk, overhead pendant or fixture)
- **Casegoods** (dresser, wardrobe, luggage rack)

### Restaurant

- **Dining seating** (chairs, banquettes, stools)
- **Dining tables** (2-top, 4-top, communal)
- **Bar** (bar stools, bar fixtures)
- **Lighting** (pendants over tables, accent lighting)

## Commercial

### Conference room

- **Conference table** (size to capacity)
- **Conference seating** (executive chairs, side chairs for guests)
- **AV and storage** (credenza for AV gear, monitor mount, whiteboard)
- **Lighting** (overhead, focal pendant if statement)

### Executive office

- **Desk** (executive desk, return / credenza)
- **Seating** (executive chair, two guest chairs)
- **Storage** (bookcase, file cabinet)
- **Lighting** (desk lamp, floor lamp, overhead)

### Open workspace

- **Workstations** (desks, benches)
- **Task seating** (Aeron / Leap / equivalents — usually configurable)
- **Collaboration zones** (lounge seating, low tables)
- **Phone booths / focus pods** (if budget allows)
- **Lighting** (overhead, task lamps per workstation)

## Residential

### Living room

- **Seating** (sofa, lounge chairs, accent chairs)
- **Tables** (coffee, side, console)
- **Lighting** (overhead, floor, table)
- **Storage** (media console, bookcase)

### Dining room

- **Dining table** (size to seating count)
- **Dining chairs**
- **Sideboard / credenza**
- **Lighting** (pendant / chandelier centered over table)

---

## Budget-allocation heuristics (rule of thumb)

These are starting points — adjust per brief. Apply only when the designer hasn't given per-category breakdowns.

- **Hospitality lobby:** Seating ~40% / Tables ~15% / Lighting ~20% / Casegoods ~20% / Accents ~5%
- **Guest room:** Bed system ~30% / Seating ~15% / Casegoods ~30% / Lighting ~15% / Tables ~10%
- **Conference room:** Table ~40% / Seating ~40% / AV ~15% / Lighting ~5%
- **Executive office:** Desk ~30% / Seating ~30% / Storage ~20% / Lighting ~10% / Accessories ~10%
